Cd-Timer !

Introduction

cd-timer is a Angular CLI component that display timer count with different options.

cd-timer is able to:

  • count up / count down.
  • manage start time and end time.
  • manage several displaying format.

Intelli format define automatically a condensed format:

  • only seconds displaying: 25s
  • minutes and seconds displaying: 02min 12s
  • hours and minutes displaying: 10h 21min
  • days and hours displaying: 2days 12min

User format:

This markup <cd-timer format="user">[hours] hours [minutes] minutes [seconds] seconds</cd-timer> will display :

[hours] hours [minutes] minutes [seconds] seconds

You have possibility to build a custom Time display with the following markup:

  • [seconds]: display seconds
  • [minutes]: display minutes
  • [hours]: display hours
  • [days]: display days

Countdown Timer

cd-timer allow to configure countdown with this markup <cd-timer countdown="true" [startTime]="100"></cd-timer>

Callbacks

There are 4 possible callbacks configuration for cd-timer:

  • (onComplete): Called when tick count reach endTime or 0
  • (onTick): Called each tick count
  • (onStart): Called when timer starts
  • (onStop): Called when timer stop
